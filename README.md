# HashLock
A deterministic password manager/generator

User can specify length of output password, character set to use, and password update counter.
    
    Character sets:
        l: abcdefghijklmnopqrstuvwxyz
        u: ABCDEFGHIJKLMNOPQRSTUVWXYZ
        d: 0123456789
        s: `~!@#$%^&*()-_=+[{]}\|;:'",<.>/?
